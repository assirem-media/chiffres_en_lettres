📱 Présentation de l'Application : Chiffres en Lettres
Chiffres en Lettres est une application web moderne (PWA) conçue pour simplifier la conversion de montants numériques en toutes lettres. C’est l’outil idéal pour la rédaction de chèques, factures, contrats et documents administratifs nécessitant une précision orthographique parfaite.

✨ Points Forts du Design & Expérience Utilisateur
🚀 Splash Screen Premium
Dès l'ouverture, l'utilisateur est accueilli par une interface de démarrage élégante :

Identité forte : Votre logo est mis en valeur sur un fond dégradé bleu nuit.
Micro-animations : Des anneaux pulsants et une lueur éthérée animent le logo.
Branding : Mention "Created by ASSIREM MEDIA" avec un dégradé de couleurs distinctif.
Rapidité : L'écran disparaît automatiquement après 2.5s ou par simple clic.
🎨 Interface "Chiffres en Lettres"
Ergonomie Calculatrice : Un clavier numérique intégré permet de saisir les montants et d'effectuer des calculs rapides (addition, multiplication, etc.).
Écran de Résultat Magnétique : Le montant en lettres s'écrit en temps réel sous la calculatrice.
Adaptabilité : Un mode "Currency" pour les devises et un mode "123" pour les nombres simples.
🛠️ Fonctionnalités Principales
1. Conversion Multi-devises
Support natif pour :

DZD (Dinar Algérien) : Gestion précise des centimes.
EUR (Euro) : Accords grammaticaux parfaits.
USD (Dollar) : Prise en charge des cents.
2. Polyglotte (Trilingue)
L'application passe instantanément d'une langue à l'autre sans recharger la page :

Français : Respect des règles complexes (ex: "quatre-vingts" vs "quatre-vingt-un").
Arabe : Support RTL (droite-gauche) avec calligraphie optimisée via la police Tajawal.
Anglais : Formatage standard international.
3. Modèles de Texte Intégrés (Templates)
Ajout automatique des mentions légales pour :

Chèque : "Payez contre ce chèque la somme de..."
Facture : "Arrêtée la présente facture à la somme de..."
Contrat : "Le montant total s'élève à..."
4. Partage Simplifié
Bouton de copie rapide pour coller le résultat directement dans vos e-mails ou documents.
Support du retour haptique (vibration) lors de la saisie sur mobile.
🏗️ Architecture Technique
Composant	Technologie	Avantage
